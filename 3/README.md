# nowcoder_JS_page
模拟牛客网的readline()实现本地调试代码
> 参考：https://github.com/zifenggao/Js-debug-page-for-nowcoder
>
> 对页面及功能进行了调整，使调试更加简洁

#### 使用说明
* 在 code -> demo.js 中编写js代码
* 打开html后，在INPUT栏插入输入文本，点击EXCUTE即可
* 修改代码后保存demo文件，直接在页面中点EXCUTE即可，不需要刷新或清缓存

* RELOAD按钮为刷新，不常用
* 提供了HJ14作为示例

* readline()和print()的使用和牛客提供的一样
